## Schema.org

```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "",
  "description": "",
  "author": {
    "@type": "Organization",
    "name": "Ambient Lounge"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Ambient Lounge",
    "url": "https://ambientlounge.ru"
  },
  "datePublished": "",
  "dateModified": "",
  "mainEntityOfPage": ""
}
```

Для FAQ-блока добавить:

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": ""
      }
    }
  ]
}
```
