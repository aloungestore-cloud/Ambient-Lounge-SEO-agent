# Audit Log — Журнал статей

Трекер всех опубликованных статей, их метрик и статуса.  
Обновлять после каждой публикации, аудита или rewrite.

---

## Статусы

| Статус | Значение |
|--------|---------|
| `draft` | черновик подготовлен, не опубликован |
| `published` | опубликован, ожидает индексации |
| `indexed` | проиндексирован в Яндекс/Google |
| `audited` | прошёл аудит метрик |
| `rewrite` | требует переработки |
| `updated` | переработан и переопубликован |
