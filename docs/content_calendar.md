# Content Calendar — Контент-план

Очередь тем для публикации и история выбора тем.  
Обновлять ежедневно после оценки тем.

---

## Статусы тем

| Статус | Значение |
|--------|---------|
| `idea` | идея, не оценена |
| `scored` | оценена по формуле, ждёт выбора |
| `selected` | выбрана для написания |
| `writing` | статья пишется |
| `ready` | статья готова, ждёт публикации |
| `published` | опубликована (перенести в audit_log.md) |
| `rejected` | отклонена с обоснованием |
